# Hide Me

This is an app that lets you hide text in images using simple steganography.

### Usage

On your machine:
```bash
git clone https://github.com/yssf-io/hide-me
cd hide-me
open index.html
```

Or try directly at [hide.yssf.io](https://hide.yssf.io).

#### Encoding

1. Drop an image
2. Write text
3. Download encoded image

#### Decoding

1. Drop an encoded image
2. Decode & read the message
